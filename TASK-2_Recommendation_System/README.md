# TASK-2: Movie Recommendation System

## 🎬 CineMatch — AI Movie Recommendation System

A lightweight movie recommendation web interface with a dark-themed UI, search input, and recommendation tabs.

## Features

- 🔍 Movie search input
- 🎛️ Recommendation and trending tabs
- 🌑 Responsive dark UI with gradients
- 📱 Mobile-friendly layout
- 🧩 Built entirely with HTML and CSS

## Project Structure

```
TASK-2_Recommendation_System/
├── index.html          # Main web interface
└── README.md           # Project documentation
```

## Technologies Used

- **Frontend**: HTML5, CSS3
- **Fonts**: Playfair Display, DM Sans (Google Fonts)

## Getting Started

1. Open `index.html` in your web browser.
2. Use the search input to explore movie titles.
3. View the interface with trending and recommendation tabs.

## Notes

- This task contains only a static frontend interface.
- No backend or database is included.
- Styles are embedded directly in `index.html`.

## Browser Support

- Chrome (Latest)
- Firefox (Latest)
- Edge (Latest)
